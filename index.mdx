Mapas Culturais é uma plataforma de software livre para mapeamento colaborativo e gestão da cultura. Usada por municípios, estados e pelo governo federal brasileiro, ela conecta agentes culturais, espaços, eventos e projetos em um único ambiente — suportando a jornada completa do fomento cultural, do cadastro à prestação de contas.

## Começando

<Steps>
  <Step title="Suba o ambiente local">
    Clone o repositório e inicie os containers Docker:
    ```bash
    git clone https://github.com/redemapas/mapas.git
    cd mapas
    docker compose up -d
    ```
    A aplicação estará disponível em `http://localhost:8080`.
  </Step>
  <Step title="Explore a plataforma">
    Acesse a interface web, crie seu perfil de agente e explore o mapa interativo com os dados culturais do seu território.
  </Step>
  <Step title="Integre via API">
    Use a API REST para consultar e integrar dados culturais em outros sistemas:
    ```bash
    curl "http://localhost:8080/api/agent/find?@select=id,name&status=EQ(1)"
    ```
  </Step>
  <Step title="Personalize a instalação">
    Crie temas, módulos e hooks para adaptar a plataforma às necessidades da sua instalação.
  </Step>
</Steps>

## Instalações na rede

Mapas Culturais está em produção em dezenas de instalações brasileiras e internacionais, incluindo:

- **SNIIC** — Sistema Nacional de Informações e Indicadores Culturais (governo federal)
- **Cultura Viva** — Programa federal de pontos de cultura
- **Rede das Artes** — Plataforma do Ministério da Cultura
- **IberculturaViva** — Rede ibero-americana de cultura viva comunitária
- Estados como Ceará, Pernambuco, Pará, Goiás, Mato Grosso e mais de 20 outros
